# refinery-stock

`refinery-stock` は `shared/stock` の陳腐化候補を棚卸しする skill です。
obsolete な知識を探索し、ユーザと相談しながら削除または更新を進めます。

## 主な責務

- `shared/stock` の obsolete 候補を洗い出す
- 削除すべきか、更新すべきか、維持すべきかを理由付きで整理する
- stock 反映済みで個別性の高い `flow` を削除候補として確認する
- `flow` の検索ノイズを、現行運用では status 追加ではなく削除優先で抑える

## 向いている場面

- `shared/stock` が増え、現在の運用に合わない知識が混ざってきたとき
- stock へ反映済みの `flow` が残っていて、検索結果のノイズになっているとき
- ユーザと会話しながら、残す知識と捨てる知識の境界を詰めたいとき
